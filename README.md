# Missing person identification 

Dataset-based missing person identification system

## What it does 
Takes a name as input 
Searches a Kaggle dataset
Displays matching person  details ( if found )

##Dataset
Source: Kaggle
Format: CSV

##Tech
Dataset handling: CSV

##Run
- Clone the repo
- Place dataset in root directory
- Run the main file
- Enter name when prompted

##Example 
Input: John Doe
Output: Matching record displayed
